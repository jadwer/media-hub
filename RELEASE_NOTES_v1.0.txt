
===========================
RELEASE NOTES – MEDIA HUB
===========================

Project........: Jaz & Gabo Media Hub
Release Version: v1.0
Release Date...: 2025-04-28

──────────────────────────
About This Release
──────────────────────────
The first public, production-ready release of the Media Hub single-page
application for easy, private sharing of audio, video, and images
between Jazmín (Jaz) and Gabino (Gabo).

──────────────────────────
Highlights
──────────────────────────
• 🎧 Unified player — automatic switch between <audio> and <video>
  for a seamless listening / viewing experience.
• 🌓 Theme switcher — Dark 🌙, Light ☀️, and Rosa-Metal 🤘 with instant
  CSS-variable theming and persistence.
• 🔍 Live search & smart ordering
  – real-time filtering + dropdown to order by:
      ‑ Date (reciente / antiguo)  
      ‑ Name (A‑Z / Z‑A)
• 💾 Persistent UI state — last search, order, and theme are stored
  in localStorage, restored on reload.
• 🏗 Skeleton loader — smooth shimmering placeholders while data
  loads or order changes.
• 🖼 Lightbox gallery for images.
• ⬇ Drag & Drop multi-upload with extension whitelisting.
• 🛠 Modular codebase — clean separation of HTML / CSS / JS / PHP.

──────────────────────────

──────────────────────────

RELEASE_NOTES_v1.0.txt ← this document

──────────────────────────
Breaking Changes
──────────────────────────
None – first stable release.

──────────────────────────
Known Issues / TODO
──────────────────────────
• Upload size limit depends on server PHP settings (upload_max_filesize).
• No password protection yet – planned for v1.1.
• Lightbox supports images only; future versions may add PDF preview.

──────────────────────────
Upgrade Path
──────────────────────────
If upgrading from earlier beta snapshots:
1. Replace all files with those from the v1.0 package.
2. Ensure /files directory retains the existing media.
3. Clear browser cache/localStorage if theme or search behaves oddly.
